# 📊 Excel for Data Analytics Portfolio

Welcome to my Excel for Data Analytics project repository! This project reflects my hands-on experience and practical coursework based on Luke Barousse's *Excel for Data Analytics* curriculum. 

Through this course, I applied core and advanced Excel methodologies to clean, analyze, transform, and visualize real-world datasets—demonstrating how spreadsheet tools can drive data-informed decisions.

---

## 🚀 Key Skills & Tools Demonstrated

* **Data Cleaning & Preparation:** Power Query, Data Transformation, M Language basics.
* **Analysis & Modeling:** Advanced Formulas (XLOOKUP, INDEX/MATCH, Logical & Text functions), Power Pivot, DAX, Data Modeling.
* **Exploratory Data Analysis:** Pivot Tables, Slicers, Calculated Fields, Summary Statistics.
* **Visualizations & Dashboards:** Dynamic Charts, Conditional Formatting, Interactive Pivot Charts.
* **Advanced Analytics & Optimization:** What-If Analysis (Goal Seek, Scenario Manager, Data Tables), Solver Add-in.
